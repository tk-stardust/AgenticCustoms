# 前端手工测试用例

## 前置条件

```bash
# 1. 启动后端（新终端）
cd backend
python main.py

# 2. 确保前端已构建
npm run build --prefix frontend

# 3. 打开浏览器访问
# http://localhost:8000
```

> 每个测试用例标注了 **[需要登录]** 或 **[无需登录]**。
